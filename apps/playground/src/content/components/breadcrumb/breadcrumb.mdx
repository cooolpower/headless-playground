# Breadcrumb

사용자가 현재 위치한 페이지의 경로를 시각적으로 나타내는 컴포넌트입니다.

Breadcrumb는 계층적 네비게이션 구조를 보여주며, 사용자가 현재 위치를 파악하고 상위 페이지로 이동할 수 있도록 도와줍니다.

### 사용법

```tsx
import { Breadcrumb } from '@repo/ui';
import type { BreadcrumbItem } from '@repo/ui';

const items: BreadcrumbItem[] = [
  { title: '홈', href: '/' },
  { title: '카테고리', href: '/category' },
  { title: '현재 페이지' },
];

export function MyComponent() {
  return <Breadcrumb items={items} />;
}
```

### API Reference

#### Breadcrumb Props

| Property     | Type                                            | Default    | Description                |
| ------------ | ----------------------------------------------- | ---------- | -------------------------- |
| items        | `BreadcrumbItem[]`                              | `[]`       | Breadcrumb 항목 배열       |
| separator    | `ReactNode`                                     | `'/'`      | 항목 구분자                |
| maxCount     | `number`                                        | -          | 최대 표시 항목 수          |
| size         | `'small' \| 'medium' \| 'large'`                | `'medium'` | Breadcrumb 크기            |
| onItemClick  | `(item: BreadcrumbItem, index: number) => void` | -          | 항목 클릭 콜백             |
| injectStyles | `boolean`                                       | `true`     | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                        | -          | 추가 CSS 클래스            |
| children     | `ReactNode`                                     | -          | 커스텀 콘텐츠              |

#### BreadcrumbItem

| Property | Type                                | Default | Description |
| -------- | ----------------------------------- | ------- | ----------- |
| title    | `ReactNode`                         | -       | 항목 제목   |
| href     | `string`                            | -       | 링크 URL    |
| onClick  | `(event: React.MouseEvent) => void` | -       | 클릭 핸들러 |
| icon     | `ReactNode`                         | -       | 항목 아이콘 |

### 특징

- **계층적 네비게이션**: 페이지 경로를 시각적으로 표현
- **아이콘 지원**: 각 항목에 아이콘 추가 가능
- **커스텀 구분자**: separator prop으로 구분자 커스터마이징 가능
- **최대 개수 제한**: maxCount prop으로 긴 경로 처리
- **링크 및 클릭 핸들러**: href 또는 onClick으로 네비게이션 처리
- **접근성**: ARIA 속성을 지원하여 스크린 리더와 호환
- **Headless 설계**: 스타일링은 사용자가 제어
