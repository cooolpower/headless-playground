# Breadcrumb

사용자가 현재 위치한 페이지의 경로를 시각적으로 나타내는 컴포넌트입니다.

Breadcrumb는 계층적 네비게이션 구조를 보여주며, 사용자가 현재 위치를 파악하고 상위 페이지로 이동할 수 있도록 도와줍니다.

### 사용법

```tsx
import { Breadcrumb } from '@cooolpower/headless-ui';
import type { BreadcrumbItem } from '@cooolpower/headless-ui';

const items: BreadcrumbItem[] = [
  { title: '홈', href: '/' },
  { title: '카테고리', href: '/category' },
  { title: '현재 페이지' },
];

export function MyComponent() {
  return <Breadcrumb items={items} />;
}
```

### API Reference

#### Breadcrumb Props

| Property     | Type                                            | Default    | Description                |
| ------------ | ----------------------------------------------- | ---------- | -------------------------- |
| items        | `BreadcrumbItem[]`                              | `[]`       | Breadcrumb 항목 배열       |
| separator    | `ReactNode`                                     | `'/'`      | 항목 구분자                |
| maxCount     | `number`                                        | -          | 최대 표시 항목 수          |
| size         | `'small' \| 'medium' \| 'large'`                | `'medium'` | Breadcrumb 크기            |
| onItemClick  | `(item: BreadcrumbItem, index: number) => void` | -          | 항목 클릭 콜백             |
| injectStyles | `boolean`                                       | `true`     | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                        | -          | 추가 CSS 클래스            |
| children     | `ReactNode`                                     | -          | 커스텀 콘텐츠              |

#### BreadcrumbItem

| Property | Type                                | Default | Description |
| -------- | ----------------------------------- | ------- | ----------- |
| title    | `ReactNode`                         | -       | 항목 제목   |
| href     | `string`                            | -       | 링크 URL    |
| onClick  | `(event: React.MouseEvent) => void` | -       | 클릭 핸들러 |
| icon     | `ReactNode`                         | -       | 항목 아이콘 |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Breadcrumb` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcBreadcrumbNav`: 전체 네비게이션 요소
> - `.hcBreadcrumbList`: `ol` 리스트 요소
> - `.hcBreadcrumbLi`: 각 항목의 `li` 요소
> - `.hcBreadcrumbItem`: 실제 링크나 텍스트를 담은 요소
> - `.hcBreadcrumbIcon`: 아이콘 요소
> - `.hcBreadcrumbSep`: 구분자 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customWrapper = style({});

// 구분자 스타일 및 간격 커스텀
globalStyle(`${customWrapper} .hcBreadcrumbSep`, {
  margin: '0 12px',
  color: '#cbd5e0',
  fontWeight: 'bold',
});

// 현재 활성화된 항목 스타일 커스텀
globalStyle(`${customWrapper} .hcBreadcrumbItem[data-current="true"]`, {
  color: '#2d3748',
  textDecoration: 'underline',
  textUnderlineOffset: '4px',
});
```
