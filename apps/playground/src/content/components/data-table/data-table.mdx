# Data Table

고급 데이터 테이블 컴포넌트입니다. 정렬, 필터링, 페이지네이션, 행 선택 등의 기능을 제공합니다.

import { Playground } from '@/components/playground/playground';
import { CssSnippet } from '@/components/playground/css-snippet';
import { DataTableCss } from '@cooolpower/headless-ui';
import {
  DemoDataTableProvider,
  DemoDataTableControls,
  DemoDataTableBasic,
  DemoDataTablePagination,
  DemoDataTableSortable,
} from './data-table.demo';

### Usage

```tsx
import { DataTable } from '@cooolpower/headless-ui';

export function MyComponent() {
  const columns = [
    { key: 'name', title: '이름' },
    { key: 'age', title: '나이' },
    { key: 'email', title: '이메일' },
  ];

  const data = [
    { name: '홍길동', age: 30, email: 'hong@example.com' },
    { name: '김철수', age: 25, email: 'kim@example.com' },
  ];

  return <DataTable columns={columns} data={data} />;
}
```

### API Reference

#### DataTable Props

| Property     | Type                   | Default | Description             |
| ------------ | ---------------------- | ------- | ----------------------- |
| value        | `any`                  | -       | 제어된 값               |
| defaultValue | `any`                  | -       | 기본값                  |
| onChange     | `(value: any) => void` | -       | 값 변경 콜백            |
| disabled     | `boolean`              | `false` | DataTable 비활성화 여부 |
| className    | `string`               | -       | 추가 CSS 클래스         |
| injectStyles | `boolean`              | `true`  | 기본 스타일 주입 여부   |
| children     | `ReactNode`            | -       | 커스텀 콘텐츠           |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`DataTable` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcDataTable`: 전체 데이터 테이블 컨테이너

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customDataTableContainer = style({});

// 데이터 테이블 컨테이너 스타일 커스텀
globalStyle(`${customDataTableContainer} .hcDataTable`, {
  backgroundColor: '#ffffff',
  border: '1px solid #e2e8f0',
  borderRadius: '8px',
  padding: '1rem',
});

// 비활성화 상태의 테이블 스타일
globalStyle(`${customDataTableContainer} .hcDataTable[data-disabled="true"]`, {
  opacity: 0.5,
  cursor: 'not-allowed',
});
```
