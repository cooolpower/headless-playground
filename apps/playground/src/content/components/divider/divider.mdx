# Divider

Provides visual separation between content sections.

### Examples

#### 기본 Divider (인터랙티브)

#### Horizontal

<Playground preview={<DemoDividerHorizontal />} />

```tsx

  <p>Content above</p>
  <Divider />
  <p>Content below</p>
</div>
```

#### With Text

<Playground preview={<DemoDividerWithText />} />

```tsx
<div>
  <p>Section A</p>
  <div style={{ display: 'flex', alignItems: 'center' }}>

#### Vertical

<Playground preview={<DemoDividerVertical />} />

```

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Divider` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcDivider`: 구분선 요소 (`hr` 또는 `div`)

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customWrapper = style({});

// 구분선 색상 및 두께 커스텀
globalStyle(`${customWrapper} .hcDivider`, {
  backgroundColor: '#e2e8f0',
  height: '2px',
  margin: '2rem 0',
});

// 수직 구분선 스타일 커스텀
globalStyle(`${customWrapper} .hcDivider[data-orientation="vertical"]`, {
  width: '2px',
  height: '24px',
  margin: '0 12px',
});
```
