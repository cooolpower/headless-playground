# Watermark

Watermark 컴포넌트는 배경에 워터마크를 표시하는 Headless 컴포넌트입니다.

## 기본 사용법

Watermark 컴포넌트를 사용하여 콘텐츠 배경에 워터마크를 표시할 수 있습니다.

<DemoWatermarkProvider>
  <Playground preview={<DemoWatermark />} controls={<WatermarkControls />} />
</DemoWatermarkProvider>

```tsx
import { Watermark } from '@repo/ui';

export function MyComponent() {
  return (
    <Watermark text="Watermark">
      <div style={{ padding: '2rem', minHeight: '400px' }}>
        Your content here
      </div>
    </Watermark>
  );
}
```

## 커스텀 워터마크

워터마크의 내용과 투명도 등을 커스터마이징할 수 있습니다.

```tsx
<Watermark text="Custom Watermark" opacity={0.15}>
  <div style={{ padding: '2rem' }}>Content with custom watermark</div>
</Watermark>
```

## API

### Watermark Props

| Property     | Description                                 | Type              | Default   |
| ------------ | ------------------------------------------- | ----------------- | --------- |
| text         | 워터마크 텍스트                             | `string`          | `'DRAFT'` |
| opacity      | 워터마크 투명도                             | `number`          | `0.1`     |
| children     | 워터마크가 적용될 콘텐츠                    | `React.ReactNode` | -         |
| className    | 추가 CSS 클래스                             | `string`          | -         |
| injectStyles | 기본 스타일을 컴포넌트가 직접 주입할지 여부 | `boolean`         | `true`    |

## 특징

- **배경 워터마크**: 콘텐츠 배경에 워터마크 표시
- **커스터마이징**: 텍스트와 투명도 등 옵션 제공
- **비침투적**: 콘텐츠를 가리지 않는 투명한 워터마크
