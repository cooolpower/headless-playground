# Timeline

시간 순서대로 이벤트나 활동을 표시하는 컴포넌트입니다. 진행 상황이나 히스토리를 시각적으로 표현합니다.

### 사용법

```tsx
import { Timeline, TimelineItem } from '@cooolpower/headless-ui';

export function MyComponent() {
  return (
    <Timeline>
      <TimelineItem label="2024-01-01">첫 번째 이벤트</TimelineItem>
      <TimelineItem label="2024-01-02">두 번째 이벤트</TimelineItem>
    </Timeline>
  );
}
```

### API Reference

#### Timeline Props

| Property     | Type                               | Default  | Description                           |
| ------------ | ---------------------------------- | -------- | ------------------------------------- |
| items        | `TimelineItem[]`                   | `[]`     | Timeline 항목 배열                    |
| mode         | `'left' \| 'alternate' \| 'right'` | `'left'` | Timeline 모드                         |
| pending      | `boolean \| ReactNode`             | -        | 대기 중 표시기 또는 콘텐츠            |
| pendingDot   | `ReactNode`                        | -        | 커스텀 대기 중 dot                    |
| reverse      | `boolean`                          | `false`  | 항목 순서 반전 여부                   |
| className    | `string`                           | -        | 추가 CSS 클래스                       |
| injectStyles | `boolean`                          | `true`   | 기본 스타일 주입 여부                 |
| children     | `ReactNode`                        | -        | 커스텀 콘텐츠 (TimelineItem 컴포넌트) |

#### TimelineItem

| Property | Type                                                                                               | Default     | Description                       |
| -------- | -------------------------------------------------------------------------------------------------- | ----------- | --------------------------------- |
| key      | `string \| number`                                                                                 | -           | 고유 키                           |
| color    | `'default' \| 'success' \| 'warning' \| 'error' \| 'info' \| 'blue' \| 'red' \| 'green' \| 'gray'` | `'default'` | Dot 색상                          |
| dot      | `ReactNode`                                                                                        | -           | 커스텀 dot 요소                   |
| label    | `ReactNode`                                                                                        | -           | 항목 레이블                       |
| children | `ReactNode`                                                                                        | -           | 항목 콘텐츠                       |
| position | `'left' \| 'right'`                                                                                | -           | 항목 위치 (mode가 alternate일 때) |

#### TimelineClassNames

내부 요소별 CSS 클래스를 지정할 수 있습니다.

| Property        | Type     | Description                     |
| --------------- | -------- | ------------------------------- |
| timeline        | `string` | Timeline 루트 요소 CSS 클래스   |
| item            | `string` | Timeline 항목 CSS 클래스        |
| itemTail        | `string` | Timeline 항목 꼬리선 CSS 클래스 |
| itemHead        | `string` | Timeline 항목 헤더 CSS 클래스   |
| dot             | `string` | Dot 요소 CSS 클래스             |
| itemContent     | `string` | Timeline 항목 콘텐츠 CSS 클래스 |
| itemLabel       | `string` | Timeline 항목 레이블 CSS 클래스 |
| itemDescription | `string` | Timeline 항목 설명 CSS 클래스   |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Timeline` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcTimeline`: 전체 타임라인 컨테이너
> - `.hcTimelineItem`: 개별 타임라인 항목
> - `.hcTimelineItemTail`: 항목 간을 연결하는 세로선 (Tail)
> - `.hcTimelineItemHead`: 점(Dot)이 위치하는 헤더 영역
> - `.hcTimelineDot`: 타임라인의 점 요소
> - `.hcTimelineItemContent`: 내용이 위치하는 영역
> - `.hcTimelineItemLabel`: 항목의 레이블(제목) 영역
> - `.hcTimelineItemDescription`: 항목의 상세 설명 영역

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customTimelineWrapper = style({});

// 연결 선(Tail) 디자인 커스텀
globalStyle(`${customTimelineWrapper} .hcTimelineItemTail`, {
  width: '4px',
  backgroundColor: '#e2e8f0',
  left: '10px',
});

// 점(Dot) 디자인 커스텀
globalStyle(`${customTimelineWrapper} .hcTimelineDot`, {
  width: '16px',
  height: '16px',
  border: '4px solid white',
  backgroundColor: '#3182ce',
  boxShadow: '0 0 0 2px #3182ce',
});

// 레이블 스타일 커스텀
globalStyle(`${customTimelineWrapper} .hcTimelineItemLabel`, {
  fontSize: '1.125rem',
  color: '#2d3748',
  fontWeight: '800',
});
```
