# Progress

작업 진행 상황을 시각적 표시기로 표시합니다.

### 사용법

```tsx
import { Progress } from '@repo/ui';

export function MyProgress() {
  return <Progress percent={75} type="line" />;
}
```

#### 인터랙티브 Progress

<Playground preview={<DemoProgressInteractive />} />

```tsx
const [progress, setProgress] = useState(50)

<Progress percent={progress} type="line" />
<button onClick={() => setProgress(Math.min(100, progress + 10))}>+10%</button>
<button onClick={() => setProgress(Math.max(0, progress - 10))}>-10%</button>
```

### API

#### Progress Props

| Property     | Type                                            | Default     | Description                |
| ------------ | ----------------------------------------------- | ----------- | -------------------------- |
| percent      | `number`                                        | `0`         | 진행률 (0-100)             |
| type         | `'line' \| 'circle'`                            | `'line'`    | Progress 타입              |
| size         | `'small' \| 'medium' \| 'large'`                | `'medium'`  | Progress 크기              |
| status       | `'normal' \| 'success' \| 'error' \| 'warning'` | `'normal'`  | Progress 상태              |
| strokeWidth  | `number`                                        | `6`         | 선 두께 (원형 타입용)      |
| strokeColor  | `string`                                        | -           | 커스텀 선 색상             |
| trailColor   | `string`                                        | `'#e5e7eb'` | 트레일 색상                |
| showInfo     | `boolean`                                       | `true`      | 진행 정보 표시 여부        |
| format       | `(percent: number) => ReactNode`                | -           | 커스텀 포맷 함수           |
| width        | `number`                                        | `120`       | 원형 Progress 너비         |
| injectStyles | `boolean`                                       | `true`      | 기본 스타일(CSS) 주입 여부 |

### Notes

- Progress는 자동으로 애니메이션 전환을 처리합니다
- 원형 Progress는 너비와 선 속성으로 완전히 커스터마이즈 가능합니다
- 선형 Progress는 색상 코딩으로 다른 상태를 지원합니다
- 커스텀 포맷 함수로 유연한 텍스트 표시가 가능합니다
- 모든 Progress 타입은 완전히 접근 가능합니다
