# Progress

작업 진행 상황을 시각적 표시기로 표시합니다.

### 사용법

```tsx
import { Progress } from '@cooolpower/headless-ui';

export function MyProgress() {
  return <Progress percent={75} type="line" />;
}
```

#### 인터랙티브 Progress

<Playground preview={<DemoProgressInteractive />} />

```tsx
const [progress, setProgress] = useState(50)

<Progress percent={progress} type="line" />
<button onClick={() => setProgress(Math.min(100, progress + 10))}>+10%</button>
<button onClick={() => setProgress(Math.max(0, progress - 10))}>-10%</button>
```

### API

#### Progress Props

| Property     | Type                                            | Default     | Description                |
| ------------ | ----------------------------------------------- | ----------- | -------------------------- |
| percent      | `number`                                        | `0`         | 진행률 (0-100)             |
| type         | `'line' \| 'circle'`                            | `'line'`    | Progress 타입              |
| size         | `'small' \| 'medium' \| 'large'`                | `'medium'`  | Progress 크기              |
| status       | `'normal' \| 'success' \| 'error' \| 'warning'` | `'normal'`  | Progress 상태              |
| strokeWidth  | `number`                                        | `6`         | 선 두께 (원형 타입용)      |
| strokeColor  | `string`                                        | -           | 커스텀 선 색상             |
| trailColor   | `string`                                        | `'#e5e7eb'` | 트레일 색상                |
| showInfo     | `boolean`                                       | `true`      | 진행 정보 표시 여부        |
| format       | `(percent: number) => ReactNode`                | -           | 커스텀 포맷 함수           |
| width        | `number`                                        | `120`       | 원형 Progress 너비         |
| injectStyles | `boolean`                                       | `true`      | 기본 스타일(CSS) 주입 여부 |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Progress` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcProgress`: 전체 프로그래스 컨테이너
> - `.hcProgressLineOuter`: 선형 프로그래스의 배경 트랙
> - `.hcProgressLineInner`: 선형 프로그래스의 진행 바
> - `.hcProgressLineText`: 선형 프로그래스의 진행률 텍스트
> - `.hcProgressCircleSvg`: 원형 프로그래스의 SVG 요소
> - `.hcProgressCircleTrail`: 원형 프로그래스의 배경 트랙 (원)
> - `.hcProgressCircleBar`: 원형 프로그래스의 진행 바 (원)
> - `.hcProgressCircleText`: 원형 프로그래스의 중앙 텍스트

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customProgressWrapper = style({});

// 선형 프로그래스 바 스타일 커스텀
globalStyle(`${customProgressWrapper} .hcProgressLineInner`, {
  background: 'linear-gradient(90deg, #4facfe 0%, #00f2fe 100%)',
  borderRadius: '4px',
  height: '12px',
});

// 프로그래스 트랙 스타일 커스텀
globalStyle(`${customProgressWrapper} .hcProgressLineOuter`, {
  backgroundColor: '#f1f5f9',
  height: '12px',
});

// 원형 프로그래스 텍스트 스타일
globalStyle(`${customProgressWrapper} .hcProgressCircleText`, {
  fontSize: '24px',
  fontWeight: '800',
  fill: '#0f172a',
});
```

### Notes

- Progress는 자동으로 애니메이션 전환을 처리합니다
- 원형 Progress는 너비와 선 속성으로 완전히 커스터마이즈 가능합니다
- 선형 Progress는 색상 코딩으로 다른 상태를 지원합니다
- 커스텀 포맷 함수로 유연한 텍스트 표시가 가능합니다
- 모든 Progress 타입은 완전히 접근 가능합니다
