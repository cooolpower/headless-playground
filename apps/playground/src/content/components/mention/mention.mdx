# Mention

사용자 멘션 기능을 제공하는 컴포넌트입니다. 텍스트 입력 시 특정 사용자나 항목을 언급할 수 있습니다.

### 사용법

```tsx
import { Mention } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [value, setValue] = useState('');

  return (
    <Mention
      value={value}
      onChange={setValue}
      options={[
        { label: 'User 1', value: 'user1' },
        { label: 'User 2', value: 'user2' },
      ]}
    />
  );
}
```

#### Load Remote Options

<Playground preview={<DemoMentionLoadRemoteOptions />} />

```tsx
const handleSearch = async (query: string): Promise<MentionOption[]> => {
  // API 호출 시뮬레이션
  return new Promise((resolve) => {
    setTimeout(() => {
      const filtered = users.filter(
        (user) =>
          user.label.toLowerCase().includes(query.toLowerCase()) ||
          user.value.toLowerCase().includes(query.toLowerCase()),
      );
      resolve(filtered);
    }, 500);
  });
};

<Mention
  value={value}
  onChange={setValue}
  onSearch={handleSearch}
  placeholder="Type @ to search users (remote)"
/>;
```

### API Reference

#### Mention Props

| Property              | Type                                                             | Default | Description                                            |
| --------------------- | ---------------------------------------------------------------- | ------- | ------------------------------------------------------ |
| value                 | `string`                                                         | -       | 제어된 값                                              |
| defaultValue          | `string`                                                         | -       | 기본값                                                 |
| onChange              | `(value: string) => void`                                        | -       | 값 변경 핸들러                                         |
| options               | `MentionOption[]`                                                | `[]`    | 멘션 옵션 목록                                         |
| onSearch              | `(query: string) => Promise<MentionOption[]> \| MentionOption[]` | -       | 원격 검색 함수 (onSearch가 제공되면 options 대신 사용) |
| disabled              | `boolean`                                                        | `false` | 비활성화 여부                                          |
| placeholder           | `string`                                                         | -       | 입력 플레이스홀더                                      |
| prefix                | `string`                                                         | `'@'`   | 멘션 접두사                                            |
| separator             | `string`                                                         | `' '`   | 멘션 구분자                                            |
| getMention            | `(option: MentionOption) => string`                              | -       | 멘션 텍스트 생성 함수                                  |
| empty                 | `ReactNode`                                                      | -       | 옵션이 없을 때 표시할 내용 (empty slot)                |
| injectStyles          | `boolean`                                                        | `true`  | 기본 스타일(CSS) 주입 여부                             |
| className             | `string`                                                         | -       | 추가 CSS 클래스                                        |
| inputWrapperClassName | `string`                                                         | -       | Input wrapper CSS 클래스                               |
| inputClassName        | `string`                                                         | -       | Input CSS 클래스                                       |
| dropdownClassName     | `string`                                                         | -       | Dropdown CSS 클래스                                    |
| optionClassName       | `string`                                                         | -       | Option CSS 클래스                                      |

#### MentionOption

| Property | Type     | Description   |
| -------- | -------- | ------------- |
| label    | `string` | 표시할 레이블 |
| value    | `string` | 옵션 값       |

### 특징

- **자동완성**: `@` 기호 입력 시 자동완성 목록 표시
- **키보드 네비게이션**: 화살표 키로 옵션 선택, Enter/Tab으로 확인
- **원격 옵션 로딩**: `onSearch` prop을 통해 비동기로 옵션을 로드할 수 있음
- **로딩 상태**: 원격 검색 중 로딩 표시
- **커스터마이징**: prefix, separator, getMention 함수로 동작 커스터마이징 가능
- **Headless 설계**: 스타일링은 사용자가 제어
- **제어/비제어 모드**: value와 defaultValue를 통한 상태 관리 지원

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Mention` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcMention`: 컴포넌트 전체 컨테이너
> - `.hcMentionInputWrap`: 입력을 감싸는 상대 위치 영역
> - `.hcMentionHighlight`: 멘션 강조 표시를 위한 배경 레이어
> - `.hcMentionHighlightMention`: 강조된 멘션 텍스트 영역
> - `.hcMentionHighlightText`: 텍스트 하이라이트 투명 영역
> - `.hcMentionTextarea`: 실제 입력을 담당하는 textarea
> - `.hcMentionDropdown`: 자동완성 드롭다운 목록
> - `.hcMentionStatus`: 드롭다운 내 상태(검색 중 등) 표시 영역
> - `.hcMentionOption`: 드롭다운 내 개별 선택 옵션

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customMentionWrapper = style({});

// 강조된 멘션 스타일 커스텀
globalStyle(`${customMentionWrapper} .hcMentionHighlightMention`, {
  backgroundColor: 'rgba(59, 130, 246, 0.2)',
  color: '#2563eb',
  fontWeight: '700',
  borderRadius: '4px',
  padding: '0 2px',
});

// 드롭다운 목록 디자인
globalStyle(`${customMentionWrapper} .hcMentionDropdown`, {
  borderRadius: '12px',
  border: '1px solid #e2e8f0',
  boxShadow: '0 10px 15px -3px rgba(0, 0, 0, 0.1)',
  marginTop: '8px',
});

// 개별 옵션 커스텀
globalStyle(`${customMentionWrapper} .hcMentionOption`, {
  padding: '10px 16px',
  transition: 'background-color 0.2s ease',
});

globalStyle(`${customMentionWrapper} .hcMentionOption:hover`, {
  backgroundColor: '#f1f5f9',
});
```
