# Date Picker

`DatePicker` 및 `DateRangePicker`는 날짜 또는 날짜 범위를 선택함에 있어 직관적이고 현대적인 인터페이스를 제공하는 컴포넌트입니다. Naive UI와 shadcn/ui의 미니멀한 Aesthetic을 계승하여 프리미엄한 사용자 경험을 제공합니다.

### 사용법

#### Single Date Picker

가장 기본적인 날짜 선택 방식입니다.

```tsx
import { DatePicker } from '@cooolpower/headless-ui';

export function SingleDatePickerDemo() {
  const [date, setDate] = useState<Date | null>(null);

  return (
    <DatePicker
      value={date}
      onChange={setDate}
      placeholder="날짜를 선택하세요"
      size="medium"
      brandColor="#eab308"
    />
  );
}
```

#### Date Range Picker

시작일과 종료일 범위를 하나의 필드에서 선택할 때 사용합니다.

```tsx
import { DateRangePicker } from '@cooolpower/headless-ui';

export function DateRangePickerDemo() {
  const [range, setRange] = useState<[Date | null, Date | null] | null>(null);

  return (
    <DateRangePicker
      value={range}
      onChange={setRange}
      size="medium"
      brandColor="#eab308"
    />
  );
}
```

### API Reference

#### DatePicker Props

| Property       | Type                             | Default               | Description               |
| :------------- | :------------------------------- | :-------------------- | :------------------------ |
| value          | `Date \| null`                   | -                     | 제어된 날짜 값            |
| defaultValue   | `Date \| null`                   | -                     | 초기 기본값               |
| onChange       | `(value: Date \| null) => void`  | -                     | 날짜 변경 시 콜백         |
| **size**       | `'small' \| 'medium' \| 'large'` | `'medium'`            | 입력창 및 달력의 크기     |
| **brandColor** | `string`                         | -                     | 포인트 컬러 (CSS 색상 값) |
| disabled       | `boolean`                        | `false`               | 컴포넌트 비활성화 여부    |
| placeholder    | `string`                         | `'날짜를 선택하세요'` | 입력창 플레이스홀더       |
| injectStyles   | `boolean`                        | `true`                | 기본 스타일 주입 여부     |

#### DateRangePicker Props

| Property       | Type                                   | Default                    | Description                 |
| :------------- | :------------------------------------- | :------------------------- | :-------------------------- |
| value          | `[Date \| null, Date \| null] \| null` | -                          | 제어된 날짜 범위 [시작, 끝] |
| onChange       | `(value: ...) => void`                 | -                          | 범위 변경 시 콜백           |
| **size**       | `'small' \| 'medium' \| 'large'`       | `'medium'`                 | 컴포넌트 전체 크기          |
| **brandColor** | `string`                               | -                          | 선택 범위 및 포인트 컬러    |
| placeholder    | `string`                               | `'날짜 범위를 선택하세요'` | 입력창 플레이스홀더         |
| injectStyles   | `boolean`                              | `true`                     | 기본 스타일 주입 여부       |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`DatePicker` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcDatePicker`: 컴포넌트 전체 컨테이너
> - `.hcDatePickerInputWrapper`: 입력창을 감싸는 영역
> - `.hcDatePickerInput`: 실제 input 요소
> - `.hcDatePickerPanelWrapper`: 달력 팝업을 감싸는 영역
> - `.hcDatePickerCalendar`: 달력 본체
> - `.hcDatePickerCalendarHeader`: 달력 상단(월/년 이동) 영역
> - `.hcDatePickerNavButton`: 이전/다음 달 이동 버튼
> - `.hcDatePickerMonthYear`: 현재 표시 중인 월/년 텍스트
> - `.hcDatePickerWeekHeader`: 요일 표시 영역
> - `.hcDatePickerDateGrid`: 날짜 그리드 영역
> - `.hcDatePickerDateCell`: 개별 날짜 셀

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customDatePickerWrapper = style({});

// 입력창 커스텀
globalStyle(`${customDatePickerWrapper} .hcDatePickerInputWrapper`, {
  borderRadius: '12px',
  border: '2px solid #e2e8f0',
  backgroundColor: '#f8fafc',
  height: '48px',
});

// 달력 패널 커스텀
globalStyle(`${customDatePickerWrapper} .hcDatePickerCalendar`, {
  borderRadius: '20px',
  border: 'none',
  boxShadow: '0 20px 25px -5px rgba(0, 0, 0, 0.1)',
  padding: '16px',
});

// 날짜 셀 스타일
globalStyle(`${customDatePickerWrapper} .hcDatePickerDateCell`, {
  borderRadius: '50%',
  fontSize: '14px',
});

// 선택된 날짜
globalStyle(`${customDatePickerWrapper} .hcDatePickerDateCellSelected`, {
  backgroundColor: '#6366f1 !important',
  color: '#fff !important',
});
```
