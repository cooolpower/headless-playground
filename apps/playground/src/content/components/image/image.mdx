# Image

이미지 표시 + 로딩/에러 상태 + (선택) 프리뷰 오버레이를 제공하는 headless 컴포넌트입니다.

### 사용법

```tsx
import { Image } from '@cooolpower/headless-ui';

export function MyImage() {
  return (
    <Image
      src="/path/to.jpg"
      alt="Example"
      width={240}
      height={160}
      preview
      placeholder={<span>Loading…</span>}
      fallback={<span>Failed</span>}
    />
  );
}
```

### API Reference

#### Image Props

| Property     | Type                                                       | Default   | Description                |
| ------------ | ---------------------------------------------------------- | --------- | -------------------------- |
| src          | `string`                                                   | -         | 이미지 URL                 |
| alt          | `string`                                                   | `''`      | 대체 텍스트                |
| width        | `number \| string`                                         | -         | 너비                       |
| height       | `number \| string`                                         | -         | 높이                       |
| objectFit    | `'fill' \| 'contain' \| 'cover' \| 'none' \| 'scale-down'` | `'cover'` | object-fit                 |
| loading      | `'lazy' \| 'eager'`                                        | `'lazy'`  | 로딩 전략                  |
| placeholder  | `ReactNode`                                                | -         | 로딩 상태 UI               |
| fallback     | `ReactNode`                                                | -         | 에러 상태 UI               |
| preview      | `boolean`                                                  | `false`   | 클릭 시 프리뷰 오버레이    |
| previewSrc   | `string`                                                   | -         | 프리뷰에서 사용할 src      |
| injectStyles | `boolean`                                                  | `true`    | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                                   | -         | 컨테이너 추가 클래스       |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Image` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcImageContainer`: 이미지를 감싸는 메인 컨테이너
> - `.hcImageImg`: 실제 `img` 요소
> - `.hcImagePlaceholder`: 로딩 중 표시되는 플레이스홀더 영역
> - `.hcImageFallback`: 에러 발생 시 표시되는 폴백 영역
> - `.hcImagePreviewOverlay`: 프리뷰 모드 시 나타나는 전체 화면 오버레이
> - `.hcImagePreviewClose`: 프리뷰 닫기 버튼
> - `.hcImagePreviewImg`: 프리뷰 오버레이 내의 확대된 이미지

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customImageWrapper = style({});

// 이미지 컨테이너 커스텀
globalStyle(`${customImageWrapper} .hcImageContainer`, {
  borderRadius: '16px',
  overflow: 'hidden',
  boxShadow: '0 4px 6px -1px rgba(0, 0, 0, 0.1)',
});

// 프리뷰 오버레이 커스텀
globalStyle(`${customImageWrapper} .hcImagePreviewOverlay`, {
  backgroundColor: 'rgba(15, 23, 42, 0.9)',
  backdropFilter: 'blur(8px)',
});

// 프리뷰 이미지 스타일
globalStyle(`${customImageWrapper} .hcImagePreviewImg`, {
  borderRadius: '8px',
  border: '4px solid #fff',
});
```
