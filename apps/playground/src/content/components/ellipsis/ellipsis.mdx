# Ellipsis

Truncates text that exceeds the container width or specified number of lines.

### Examples

#### 기본 Ellipsis (인터랙티브)

<DemoEllipsisBasicProvider>
  <Playground
    preview={<DemoEllipsisBasicWithControls />}
    controls={<DemoEllipsisBasicControls />}
  />
</DemoEllipsisBasicProvider>

#### Single Line

<Playground preview={<DemoEllipsisSingleLine />} />

```tsx
<Ellipsis>
  <div>Long text that will be truncated...</div>
</Ellipsis>
```

#### Multi-line

<Playground preview={<DemoEllipsisMultiLine />} />

```tsx
<Ellipsis lines={3}>
  <div>Long text that will be truncated after 3 lines...</div>
</Ellipsis>
```

#### Short Text

<Playground preview={<DemoEllipsisShortText />} />

```tsx
<Ellipsis>
  <div>Short text that fits</div>
</Ellipsis>
```

#### Different Sizes

<Playground preview={<DemoEllipsisSizes />} />

```tsx
<div style={{ width: '200px' }}>
  <Ellipsis>
    <div>Text in different container sizes</div>
  </Ellipsis>
</div>
```

### API

| Property     | Description                                 | Type      | Default |
| ------------ | ------------------------------------------- | --------- | ------- |
| lines        | 줄 수 (미지정/1이면 single-line)            | `number`  | `1`     |
| className    | 추가 CSS 클래스                             | `string`  | -       |
| injectStyles | 기본 스타일을 컴포넌트가 직접 주입할지 여부 | `boolean` | `true`  |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Ellipsis` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcEllipsisSingle`: 단일 행 생략 시 적용되는 클래스
> - `.hcEllipsisMulti`: 다중 행 생략 시 적용되는 클래스

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customEllipsisWrapper = style({});

// 단일 행 생략 스타일 커스텀
globalStyle(`${customEllipsisWrapper} .hcEllipsisSingle`, {
  maxWidth: '200px',
  color: '#4a5568',
});

// 다중 행 생략 스타일 커스텀 (예: 3줄 제한)
globalStyle(`${customEllipsisWrapper} .hcEllipsisMulti`, {
  display: '-webkit-box',
  WebkitBoxOrient: 'vertical',
  WebkitLineClamp: 3,
  overflow: 'hidden',
  lineHeight: '1.6',
});
```
