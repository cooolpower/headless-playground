### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Card` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcCard`: 전체 카드 컨테이너
> - `.hcCardHeader`: 헤더 섹션
> - `.hcCardContent`: 메인 콘텐츠 섹션
> - `.hcCardFooter`: 푸터 섹션

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customWrapper = style({});

// 카드 테두리 및 그림자 커스텀
globalStyle(`${customWrapper} .hcCard`, {
  border: 'none',
  boxShadow: '0 10px 25px rgba(0, 0, 0, 0.1)',
  borderRadius: '20px',
});

// 헤더 영역 스타일 커스텀
globalStyle(`${customWrapper} .hcCardHeader`, {
  backgroundColor: '#f7fafc',
  padding: '1.5rem',
  borderBottom: '1px solid #edf2f7',
});
```
