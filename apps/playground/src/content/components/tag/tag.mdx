### API Reference

#### Tag Props

| Property     | Type                                                          | Default     | Description                |
| ------------ | ------------------------------------------------------------- | ----------- | -------------------------- |
| children     | `ReactNode`                                                   | -           | 레이블 콘텐츠              |
| variant      | `'default' \| 'primary' \| 'success' \| 'warning' \| 'error'` | `'default'` | 태그의 스타일 종류         |
| size         | `'small' \| 'medium' \| 'large'`                              | `'medium'`  | 태그 크기                  |
| closable     | `boolean`                                                     | `false`     | 닫기 아이콘 표시 여부      |
| onClose      | `() => void`                                                  | -           | 닫기 아이콘 클릭 시 콜백   |
| disabled     | `boolean`                                                     | `false`     | 비활성화 여부              |
| injectStyles | `boolean`                                                     | `true`      | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                                      | -           | 추가 CSS 클래스            |
| style        | `React.CSSProperties`                                         | -           | 인라인 스타일              |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Tag` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcTag`: 태그 전체 요소
> - `.hcTagClose`: 닫기(X) 버튼 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customTagWrapper = style({});

// 태그 모양 및 색상 커스텀
globalStyle(`${customTagWrapper} .hcTag`, {
  backgroundColor: '#ebf8ff',
  color: '#2b6cb0',
  border: '1px solid #90cdf4',
  borderRadius: '9999px',
  padding: '4px 12px',
});

// 닫기 버튼 스타일 커스텀
globalStyle(`${customTagWrapper} .hcTagClose`, {
  marginLeft: '8px',
  color: '#4299e1',
  transition: 'color 0.2s',
});

globalStyle(`${customTagWrapper} .hcTagClose:hover`, {
  color: '#2b6cb0',
});
```
