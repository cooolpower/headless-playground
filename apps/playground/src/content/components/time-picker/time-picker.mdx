# Time Picker

시간을 선택할 수 있는 입력 컴포넌트입니다. 시, 분, 초를 선택할 수 있습니다.

### 사용법

```tsx
import { TimePicker } from '@headless/ui';

export function MyComponent() {
  const [time, setTime] = useState<Date | null>(null);

  return (
    <TimePicker
      value={time}
      onChange={setTime}
      placeholder="시간을 선택하세요"
    />
  );
}
```

### API Reference

#### TimePicker Props

| Property                | Type                                                         | Default      | Description              |
| ----------------------- | ------------------------------------------------------------ | ------------ | ------------------------ |
| value                   | `Date`                                                       | -            | 제어된 선택 시간         |
| defaultValue            | `Date`                                                       | -            | 기본 선택 시간           |
| onChange                | `(time: Date \| null, timeString: string) => void`           | -            | 시간 변경 콜백           |
| onSelect                | `(time: Date) => void`                                       | -            | 시간 선택 콜백           |
| format                  | `string`                                                     | `'HH:mm:ss'` | 시간 포맷 문자열         |
| placeholder             | `string`                                                     | -            | 입력 플레이스홀더        |
| disabled                | `boolean`                                                    | `false`      | Picker 비활성화 여부     |
| allowClear              | `boolean`                                                    | `false`      | 지우기 버튼 표시 여부    |
| showNow                 | `boolean`                                                    | `false`      | "지금" 버튼 표시 여부    |
| size                    | `'small' \| 'medium' \| 'large'`                             | `'medium'`   | Picker 크기              |
| use12Hours              | `boolean`                                                    | `false`      | 12시간 형식 사용 여부    |
| hourStep                | `number`                                                     | `1`          | 시간 선택 단계           |
| minuteStep              | `number`                                                     | `1`          | 분 선택 단계             |
| secondStep              | `number`                                                     | `1`          | 초 선택 단계             |
| disabledHours           | `() => number[]`                                             | -            | 특정 시간 비활성화 함수  |
| disabledMinutes         | `(selectedHour: number) => number[]`                         | -            | 특정 분 비활성화 함수    |
| disabledSeconds         | `(selectedHour: number, selectedMinute: number) => number[]` | -            | 특정 초 비활성화 함수    |
| className               | `string`                                                     | -            | 추가 CSS 클래스          |
| panelClassName          | `string`                                                     | -            | 패널 CSS 클래스          |
| inputWrapperClassName   | `string`                                                     | -            | Input wrapper CSS 클래스 |
| inputClassName          | `string`                                                     | -            | Input CSS 클래스         |
| inputActionsClassName   | `string`                                                     | -            | Input actions CSS 클래스 |
| clearButtonClassName    | `string`                                                     | -            | 지우기 버튼 CSS 클래스   |
| dropdownButtonClassName | `string`                                                     | -            | 드롭다운 버튼 CSS 클래스 |
| backdropClassName       | `string`                                                     | -            | 백드롭 CSS 클래스        |
| panelWrapperClassName   | `string`                                                     | -            | 패널 wrapper CSS 클래스  |
| children                | `ReactNode`                                                  | -            | 커스텀 콘텐츠            |
| injectStyles            | `boolean`                                                    | `true`       | 기본 스타일 주입 여부    |
