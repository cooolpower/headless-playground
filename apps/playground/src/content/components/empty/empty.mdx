# Empty

빈 상태를 표시하는 컴포넌트입니다. 데이터가 없을 때 사용자에게 명확한 피드백을 제공합니다.

### Usage

```tsx
import { Empty } from '@cooolpower/headless-ui';

export function MyComponent() {
  return <Empty description="데이터가 없습니다" />;
}
```

### API Reference

#### Empty Props

| Property     | Type            | Default               | Description                             |
| ------------ | --------------- | --------------------- | --------------------------------------- |
| image        | `ReactNode`     | `<DefaultImage />`    | Custom empty image                      |
| description  | `ReactNode`     | `'데이터가 없습니다'` | Empty description text                  |
| children     | `ReactNode`     | -                     | Footer content (usually action buttons) |
| imageStyle   | `CSSProperties` | -                     | Custom styles for image container       |
| injectStyles | `boolean`       | `true`                | 기본 스타일(CSS) 주입 여부              |
| className    | `string`        | -                     | Additional CSS class                    |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Empty` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcEmpty`: 전체 빈 상태 컨테이너
> - `.hcEmptyImage`: 이미지/아이콘 영역
> - `.hcEmptyDesc`: 설명 텍스트 영역
> - `.hcEmptyFooter`: 하단 추가 콘텐츠 영역 (버튼 등)

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customEmptyWrapper = style({});

// 빈 상태 컨테이너 스타일 커스텀
globalStyle(`${customEmptyWrapper} .hcEmpty`, {
  padding: '60px 0',
  backgroundColor: '#f8fafc',
  borderRadius: '16px',
});

// 설명 텍스트 스타일 커스텀
globalStyle(`${customEmptyWrapper} .hcEmptyDesc`, {
  fontSize: '1.125rem',
  color: '#64748b',
  marginTop: '1.5rem',
});

// 이미지(아이콘) 스타일 커스텀
globalStyle(`${customEmptyWrapper} .hcEmptyImage`, {
  fontSize: '64px',
  filter: 'grayscale(100%)',
  opacity: 0.5,
});
```
