# Page Header

Page Header는 페이지의 **제목**, **부제목**, **액션 영역(extra)** 등을 배치하기 위한 Headless 컴포넌트입니다.

## 기본 사용법

Page Header 컴포넌트를 사용하여 페이지 상단에 헤더를 표시할 수 있습니다.

<DemoPageHeaderProvider>
  <Playground
    preview={<DemoPageHeaderWithControls />}
    controls={<DemoPageHeaderControls />}
  />
</DemoPageHeaderProvider>

```tsx
import { PageHeader } from '@headless/ui';
import { Button } from '@headless/ui';

export function MyComponent() {
  return (
    <PageHeader
      title="Page Title"
      subtitle="Page subtitle"
      extra={<Button>Action</Button>}
    />
  );
}
```

## 액션 영역(extra)

extra prop을 통해 헤더에 액션 버튼을 추가할 수 있습니다.

```tsx
<PageHeader
  title="Page Title"
  extra={
    <>
      <Button>Secondary</Button>
      <Button>Primary</Button>
    </>
  }
/>
```

## API

### PageHeader Props

| Property     | Description                                 | Type              | Default |
| ------------ | ------------------------------------------- | ----------------- | ------- |
| title        | 페이지 제목                                 | `React.ReactNode` | -       |
| subtitle     | 페이지 부제목                               | `React.ReactNode` | -       |
| extra        | 헤더 오른쪽에 표시할 추가 콘텐츠            | `React.ReactNode` | -       |
| className    | 추가 CSS 클래스                             | `string`          | -       |
| injectStyles | 기본 스타일을 컴포넌트가 직접 주입할지 여부 | `boolean`         | `true`  |
