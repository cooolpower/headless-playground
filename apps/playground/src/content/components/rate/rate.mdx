# Rate

별점 평가를 위한 컴포넌트입니다. 사용자가 항목에 대한 평점을 선택할 수 있습니다.

### 사용법

```tsx
import { Rate } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [value, setValue] = useState(0);

  return <Rate value={value} onChange={setValue} count={5} />;
}
```

### API Reference

#### Rate Props

| Property     | Type                      | Default | Description           |
| ------------ | ------------------------- | ------- | --------------------- |
| count        | `number`                  | `5`     | 전체 별 개수          |
| value        | `number`                  | -       | 제어된 값             |
| defaultValue | `number`                  | `0`     | 기본값                |
| onChange     | `(value: number) => void` | -       | 값 변경 콜백          |
| allowHalf    | `boolean`                 | `false` | 반 별점 허용 여부     |
| disabled     | `boolean`                 | `false` | Rate 비활성화 여부    |
| character    | `ReactNode`               | `'★'`   | 표시할 커스텀 문자    |
| className    | `string`                  | -       | 추가 CSS 클래스       |
| injectStyles | `boolean`                 | `true`  | 기본 스타일 주입 여부 |
| children     | `ReactNode`               | -       | 커스텀 콘텐츠         |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Rate` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcRate`: 전체 별점 컨테이너
> - `.hcRateStarWrapper`: 개별 별점(또는 문자)을 감싸는 컨테이너
> - `.hcRateStar`: 기본 별점 요소
> - `.hcRateStarHalf`: 반 개 별점 표시를 위한 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customRateWrapper = style({});

// 별점 크기 및 간격 조정
globalStyle(`${customRateWrapper} .hcRate`, {
  gap: '8px',
  fontSize: '32px',
});

// 기본 별점 색상 (비활성 상태)
globalStyle(`${customRateWrapper} .hcRateStar`, {
  color: '#e2e8f0',
});

// 활성화된 별점 색상 (선택된 상태)
globalStyle(`${customRateWrapper} .hcRateStar[data-active="true"]`, {
  color: '#f6ad55',
});

// 반 별점 색상 커스텀
globalStyle(`${customRateWrapper} .hcRateStarHalf[data-half-active="true"]`, {
  color: '#f6ad55',
});
```
