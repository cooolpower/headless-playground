# Loading Bar

페이지 상단에 표시되는 로딩 진행 바 컴포넌트입니다. 비동기 작업의 진행 상황을 시각적으로 표시합니다.

### 사용법

```tsx
import { LoadingBar } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [percent, setPercent] = useState(0);

  return <LoadingBar percent={percent} show={true} color="#007bff" />;
}
```

### API Reference

#### LoadingBar Props

| Property     | Type        | Default     | Description                |
| ------------ | ----------- | ----------- | -------------------------- |
| percent      | `number`    | -           | 로딩 진행률 (0-100)        |
| show         | `boolean`   | -           | Loading Bar 표시 여부      |
| color        | `string`    | `'#007bff'` | Loading Bar 색상           |
| height       | `number`    | `3`         | Loading Bar 높이 (픽셀)    |
| injectStyles | `boolean`   | `true`      | 기본 스타일(CSS) 주입 여부 |
| className    | `string`    | -           | 추가 CSS 클래스            |
| children     | `ReactNode` | -           | 커스텀 콘텐츠              |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`LoadingBar` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcLoadingBar`: 로딩 바의 전체 컨테이너
> - `.hcLoadingBarProgress`: 진행률을 나타내는 바 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customLoadingBarWrapper = style({});

// 전체 컨테이너 스타일 커스텀
globalStyle(`${customLoadingBarWrapper} .hcLoadingBar`, {
  height: '6px',
  backgroundColor: 'rgba(241, 245, 249, 0.5)',
  backdropFilter: 'blur(4px)',
});

// 진행 바 디자인 커스텀 (그라데이션 효과)
globalStyle(`${customLoadingBarWrapper} .hcLoadingBarProgress`, {
  background: 'linear-gradient(90deg, #6366f1 0%, #a855f7 100%)',
  boxShadow: '0 0 8px rgba(99, 102, 241, 0.5)',
});
```
