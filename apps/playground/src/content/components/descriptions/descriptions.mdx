# Descriptions

설명 목록을 표시하는 컴포넌트입니다. 키-값 쌍의 정보를 구조화된 형태로 보여줍니다.

### Usage

```tsx
import {
  Descriptions,
  DescriptionsItem,
} from '@headless/ui';

export function MyComponent() {
  return (
    <Descriptions title="사용자 정보">
      <DescriptionsItem label="이름">홍길동</DescriptionsItem>
      <DescriptionsItem label="이메일">hong@example.com</DescriptionsItem>
      <DescriptionsItem label="전화번호">010-1234-5678</DescriptionsItem>
    </Descriptions>
  );
}
```

### API Reference

#### Descriptions Props

| Property     | Type                             | Default        | Description                               |
| ------------ | -------------------------------- | -------------- | ----------------------------------------- |
| title        | `ReactNode`                      | -              | Descriptions 제목                         |
| items        | `DescriptionsItem[]`             | `[]`           | 항목 배열                                 |
| column       | `number \| ResponsiveConfig`     | `3`            | 컬럼 개수                                 |
| size         | `'small' \| 'medium' \| 'large'` | `'medium'`     | Descriptions 크기                         |
| bordered     | `boolean`                        | `false`        | 테두리 표시 여부                          |
| layout       | `'horizontal' \| 'vertical'`     | `'horizontal'` | 레이아웃 방향                             |
| colon        | `boolean`                        | `true`         | 레이블 뒤 콜론 표시 여부                  |
| className    | `string`                         | -              | 추가 CSS 클래스                           |
| injectStyles | `boolean`                        | `true`         | 기본 스타일 주입 여부                     |
| children     | `ReactNode`                      | -              | 커스텀 콘텐츠 (DescriptionsItem 컴포넌트) |

#### DescriptionsItem

| Property | Type               | Default | Description      |
| -------- | ------------------ | ------- | ---------------- |
| key      | `string \| number` | -       | 고유 키          |
| label    | `ReactNode`        | -       | 항목 레이블      |
| children | `ReactNode`        | -       | 항목 콘텐츠      |
| span     | `number`           | `1`     | 병합할 컬럼 개수 |
