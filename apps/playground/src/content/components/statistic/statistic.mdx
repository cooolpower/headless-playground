# Statistic

통계 숫자를 표시하는 컴포넌트입니다. 숫자 포맷팅, prefix/suffix, 로딩 상태 등을 지원합니다.

### 사용법

```tsx
import { Statistic } from '@cooolpower/headless-ui';

export function MyComponent() {
  return <Statistic title="총 사용자" value={1234567} suffix="명" />;
}
```

#### 애니메이션 값

숫자가 0부터 목표 값까지 롤링 애니메이션으로 증가하는 예제입니다.

<Playground preview={<DemoStatisticAnimated />} />

```tsx
function useAnimatedValue(targetValue: number, duration: number = 2000) {
  const [currentValue, setCurrentValue] = useState(0);

  useEffect(() => {
    const startValue = 0;
    const endValue = targetValue;
    const startTime = Date.now();

    const animate = () => {
      const elapsed = Date.now() - startTime;
      const progress = Math.min(elapsed / duration, 1);

      // Easing function (ease-out)
      const easeOut = 1 - Math.pow(1 - progress, 3);

      const newValue = Math.floor(
        startValue + (endValue - startValue) * easeOut,
      );
      setCurrentValue(newValue);

      if (progress < 1) {
        requestAnimationFrame(animate);
      } else {
        setCurrentValue(endValue);
      }
    };

    const animationId = requestAnimationFrame(animate);

    return () => {
      cancelAnimationFrame(animationId);
    };
  }, [targetValue, duration]);

  return currentValue;
}

export function MyComponent() {
  const [targetValue, setTargetValue] = useState(1234567);
  const animatedValue = useAnimatedValue(targetValue, 2000);

  return <Statistic title="총 사용자" value={animatedValue} />;
}
```

### API Reference

#### Statistic Props

| Property         | Type                  | Default | Description                 |
| ---------------- | --------------------- | ------- | --------------------------- |
| title            | `ReactNode`           | -       | 제목 텍스트                 |
| value            | `string \| number`    | -       | 표시할 값                   |
| prefix           | `ReactNode`           | -       | 접두사 요소 (예: 통화 기호) |
| suffix           | `ReactNode`           | -       | 접미사 요소 (예: 단위)      |
| precision        | `number`              | `2`     | 소수점 자릿수               |
| groupSeparator   | `string`              | `','`   | 천 단위 구분자              |
| decimalSeparator | `string`              | `'.'`   | 소수점 구분자               |
| loading          | `boolean`             | `false` | 로딩 상태 표시 여부         |
| valueStyle       | `CSSProperties`       | -       | 값에 대한 커스텀 스타일     |
| injectStyles     | `boolean`             | `true`  | 기본 스타일(CSS) 주입 여부  |
| className        | `string`              | -       | 추가 CSS 클래스             |
| classNames       | `StatisticClassNames` | -       | 내부 요소에 대한 CSS 클래스 |
| children         | `ReactNode`           | -       | 커스텀 콘텐츠               |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Statistic` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcStatistic`: 전체 통계 컨테이너
> - `.hcStatisticTitle`: 통계 제목 영역
> - `.hcStatisticContent`: 값과 prefix/suffix를 포함하는 영역
> - `.hcStatisticValue`: 실제 숫자 값 영역
> - `.hcStatisticPrefix`: 접두사 영역
> - `.hcStatisticSuffix`: 접미사 영역
> - `.hcStatisticSkeleton`: 로딩 시 표시되는 스켈레톤

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customStatisticWrapper = style({});

// 통계 제목 스타일 커스텀
globalStyle(`${customStatisticWrapper} .hcStatisticTitle`, {
  fontSize: '0.875rem',
  textTransform: 'uppercase',
  letterSpacing: '0.05em',
  color: '#718096',
});

// 통계 값 스타일 커스텀
globalStyle(`${customStatisticWrapper} .hcStatisticValue`, {
  fontSize: '3rem',
  fontWeight: '900',
  color: '#2d3748',
  fontFamily: 'monospace',
});

// 접미사 스타일
globalStyle(`${customStatisticWrapper} .hcStatisticSuffix`, {
  fontSize: '1.25rem',
  marginLeft: '4px',
  color: '#a0aec0',
});
```
