# Typography

일관된 계층 구조와 스타일링을 가진 시맨틱 텍스트 요소를 제공합니다. Typography 컴포넌트는 애플리케이션 전체에서 시각적 일관성과 적절한 텍스트 구조를 유지하는 데 도움이 됩니다.

import { CssSnippet } from '@/components/playground/css-snippet';
import { TypographyCss } from '@headless/ui';

### 사용법

```tsx
import {
  Title,
  Text,
  Paragraph,
} from '@headless/ui';

export function MyComponent() {
  return (
    <>
      <Title level={1}>메인 제목</Title>
      <Paragraph>콘텐츠 단락</Paragraph>
      <Text>인라인 텍스트</Text>
    </>
  );
}
```

### API Reference

#### Title Props

| Property     | Type                         | Default | Description                |
| ------------ | ---------------------------- | ------- | -------------------------- |
| level        | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | `1`     | 제목 레벨                  |
| injectStyles | `boolean`                    | `true`  | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                     | -       | 추가 CSS 클래스            |
| children     | `ReactNode`                  | -       | 제목 콘텐츠                |

#### Text Props

| Property     | Type        | Default | Description                |
| ------------ | ----------- | ------- | -------------------------- |
| injectStyles | `boolean`   | `true`  | 기본 스타일(CSS) 주입 여부 |
| className    | `string`    | -       | 추가 CSS 클래스            |
| children     | `ReactNode` | -       | 텍스트 콘텐츠              |

#### Paragraph Props

| Property     | Type        | Default | Description                |
| ------------ | ----------- | ------- | -------------------------- |
| injectStyles | `boolean`   | `true`  | 기본 스타일(CSS) 주입 여부 |
| className    | `string`    | -       | 추가 CSS 클래스            |
| children     | `ReactNode` | -       | 단락 콘텐츠                |

### Best Practices

- 시맨틱 HTML 구조를 유지하기 위해 제목에는 `Title` 사용
- 본문 텍스트 블록에는 `Paragraph` 사용
- 일관된 스타일링이 필요한 인라인 텍스트에는 `Text` 사용
- 적절한 제목 계층 구조 유지 (레벨 건너뛰지 않기)
- 풍부한 텍스트 레이아웃을 만들기 위해 컴포넌트 결합
