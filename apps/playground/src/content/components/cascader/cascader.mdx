# Cascader

계층적 선택 컴포넌트입니다. Input을 클릭하면 1Depth 패널이 나타나고, 옵션을 선택하면 2Depth 패널이 옆으로 나타나는 방식으로 동작합니다.

Cascader는 계층적 데이터 구조에서 값을 선택할 수 있는 컴포넌트입니다. Naive UI의 Input Group 예제와 유사한 방식으로 동작합니다.

### 사용법

```tsx
import { Cascader } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [value, setValue] = useState<string[]>([]);

  const options = [
    {
      label: '프론트엔드',
      value: 'frontend',
      children: [
        { label: 'React', value: 'react' },
        { label: 'Vue', value: 'vue' },
      ],
    },
  ];

  return (
    <Cascader
      options={options}
      value={value}
      onChange={setValue}
      placeholder="선택하세요..."
    />
  );
}
```

### API Reference

#### Cascader Props

| Property       | Type                             | Default           | Description           |
| -------------- | -------------------------------- | ----------------- | --------------------- |
| options        | `CascaderOption[]`               | `[]`              | 계층적 옵션 데이터    |
| value          | `string[]`                       | -                 | 제어된 값             |
| defaultValue   | `string[]`                       | `[]`              | 기본 값               |
| placeholder    | `string`                         | `'선택하세요...'` | Placeholder 텍스트    |
| disabled       | `boolean`                        | `false`           | 비활성화 여부         |
| separator      | `string`                         | `' / '`           | 선택된 값들의 구분자  |
| size           | `'small' \| 'medium' \| 'large'` | `'medium'`        | 크기                  |
| onChange       | `(value: string[]) => void`      | -                 | 값 변경 핸들러        |
| onOpen         | `() => void`                     | -                 | 패널 열림 핸들러      |
| onClose        | `() => void`                     | -                 | 패널 닫힘 핸들러      |
| injectStyles   | `boolean`                        | `true`            | 기본 스타일 주입 여부 |
| className      | `string`                         | -                 | 추가 CSS 클래스       |
| inputClassName | `string`                         | -                 | Input 요소 CSS 클래스 |

#### CascaderOption

| Property | Type               | Description   |
| -------- | ------------------ | ------------- |
| label    | `string`           | 옵션 레이블   |
| value    | `string \| number` | 옵션 값       |
| disabled | `boolean`          | 비활성화 여부 |
| children | `CascaderOption[]` | 하위 옵션들   |

### 특징

- **계층적 선택**: 1Depth와 2Depth 패널을 통한 계층적 선택 지원
- **체크박스 표시**: 각 옵션에 체크박스 표시로 선택 상태 명확히 표시
- **구분자 커스터마이징**: 선택된 값들의 구분자를 사용자가 설정 가능
- **제어/비제어 모드**: value와 defaultValue를 통한 상태 관리 지원
- **크기 조절**: small, medium, large 크기 지원
- **Headless 설계**: 스타일링은 사용자가 제어
- **접근성**: ARIA 속성 지원

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Cascader` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcCascader`: 컴포넌트 전체 컨테이너
> - `.hcCascaderTrigger`: 선택된 값을 표시하는 트리거 영역
> - `.hcCascaderArrow`: 트리거 내 화살표 아이콘 영역
> - `.hcCascaderArrowIcon`: 화살표 아이콘 자체
> - `.hcCascaderPanels`: 옵션 패널들이 배치되는 컨테이너
> - `.hcCascaderPanel`: 개별 레벨의 옵션 패널
> - `.hcCascaderOption`: 패널 내 개별 옵션 항목
> - `.hcCascaderOptionMain`: 옵션의 메인 콘텐츠 영역(레이블 포함)
> - `.hcCascaderOptionLabel`: 옵션의 텍스트 레이블
> - `.hcCascaderNextIcon`: 하위 메뉴가 있음을 나타내는 아이콘

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customCascaderWrapper = style({});

// 패널 컨테이너 스타일 커스텀
globalStyle(`${customCascaderWrapper} .hcCascaderPanels`, {
  backgroundColor: '#ffffff',
  borderRadius: '12px',
  boxShadow:
    '0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)',
  border: '1px solid #e2e8f0',
  marginTop: '8px',
});

// 개별 옵션 스타일 커스텀
globalStyle(`${customCascaderWrapper} .hcCascaderOption`, {
  padding: '10px 16px',
  fontSize: '14px',
  transition: 'all 0.2s ease',
});

// 활성화/선택된 옵션 스타일
globalStyle(`${customCascaderWrapper} .hcCascaderOption[data-active="true"]`, {
  backgroundColor: '#f1f5f9',
});

globalStyle(
  `${customCascaderWrapper} .hcCascaderOption[data-selected="true"]`,
  {
    backgroundColor: '#eff6ff',
    color: '#3b82f6',
    fontWeight: '600',
  },
);

// 다음 아이콘 스타일
globalStyle(`${customCascaderWrapper} .hcCascaderNextIcon`, {
  color: '#94a3b8',
  fontSize: '12px',
});
```
