# Badge

UI 요소에 작은 숫자 값이나 상태 설명자를 표시합니다.

### 사용법

```tsx
import { Badge } from '@cooolpower/headless-ui';

export function MyComponent() {
  return (
    <Badge count={5}>
      <button>메시지</button>
    </Badge>
  );
}
```

### API Reference

#### Badge Props

| Property        | Type                                        | Default     | Description                    |
| --------------- | ------------------------------------------- | ----------- | ------------------------------ |
| children        | `ReactNode`                                 | -           | Badge를 첨부할 요소            |
| count           | `number \| string`                          | -           | Badge에 표시할 숫자            |
| dot             | `boolean`                                   | `false`     | 숫자 대신 점 표시 여부         |
| showZero        | `boolean`                                   | `false`     | count가 0일 때 Badge 표시 여부 |
| maxCount        | `number`                                    | `99`        | 최대 표시 숫자                 |
| color           | `BadgeColor \| string`                      | `'default'` | Badge 색상                     |
| size            | `BadgeSize`                                 | `'medium'`  | Badge 크기                     |
| processing      | `boolean`                                   | `false`     | 처리 중 애니메이션 표시 여부   |
| backgroundColor | `string`                                    | -           | 배지의 커스텀 배경색           |
| textColor       | `string`                                    | -           | 배지의 커스텀 텍스트 색상      |
| borderWidth     | `string`                                    | -           | 배지의 테두리 두께             |
| borderColor     | `string`                                    | -           | 배지의 테두리 색상             |
| borderStyle     | `'none' \| 'solid' \| 'dashed' \| 'dotted'` | -           | 배지의 테두리 스타일           |
| injectStyles    | `boolean`                                   | `true`      | 기본 스타일(CSS) 주입 여부     |
| className       | `string`                                    | -           | 커스텀 CSS 클래스              |

#### BadgeColor

```tsx
type BadgeColor =

  | 'success'
  | 'warning'
  | 'danger'
  | 'info';
```

#### BadgeSize


### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Badge` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcBadgeContainer`: 배지와 자식 요소를 감싸는 전체 컨테이너
> - `.hcBadgeCount`: 숫자가 표시되는 배지 요소
> - `.hcBadgeDot`: 점으로 표시되는 배지 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customWrapper = style({});

// 배지 위치 및 크기 커스텀
globalStyle(`${customWrapper} .hcBadgeCount`, {
  backgroundColor: '#553c9a',
  color: '#fff',
  fontSize: '0.8rem',
  top: '4px',
  right: '4px',
});

// 점 스타일 커스텀
globalStyle(`${customWrapper} .hcBadgeDot`, {
  width: '12px',
  height: '12px',
  border: '2px solid white',
});
```
