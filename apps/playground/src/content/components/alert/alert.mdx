# Alert

중요한 메시지를 다양한 타입과 스타일로 사용자에게 표시합니다.

### 사용법

```tsx
import { Alert } from '@headless/ui';

export function MyAlert() {
  return (
    <Alert
      type="success"
      title="성공!"
      description="작업이 성공적으로 완료되었습니다."
    />
  );
}
```

### API

#### Alert Props

| Property     | Type                                          | Default    | Description                |
| ------------ | --------------------------------------------- | ---------- | -------------------------- |
| type         | `'success' \| 'error' \| 'warning' \| 'info'` | `'info'`   | Alert 타입                 |
| showIcon     | `boolean`                                     | `false`    | 아이콘 표시 여부           |
| closable     | `boolean`                                     | `false`    | 닫기 가능 여부             |
| onClose      | `() => void`                                  | -          | Alert가 닫힐 때 콜백       |
| title        | `ReactNode`                                   | -          | Alert 제목                 |
| description  | `ReactNode`                                   | -          | Alert 설명                 |
| banner       | `boolean`                                     | `false`    | 배너로 표시 여부           |
| size         | `'small' \| 'medium' \| 'large'`              | `'medium'` | Alert 크기                 |
| injectStyles | `boolean`                                     | `true`     | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                      | -          | 추가 CSS 클래스            |

### Notes

- Alert는 타입에 따라 의미론적 색상 코딩을 사용합니다
- 배너 Alert는 전체 너비를 차지하며 일반적으로 시스템 전체 메시지에 사용됩니다
- 닫기 가능한 Alert는 사용자가 메시지를 해제할 수 있도록 합니다
- `showIcon`이 true일 때 Alert 타입에 따라 아이콘이 자동으로 선택됩니다
