# Alert

중요한 메시지를 다양한 타입과 스타일로 사용자에게 표시합니다.

### 사용법

```tsx
import { Alert } from '@cooolpower/headless-ui';

export function MyAlert() {
  return (
    <Alert
      type="success"
      title="성공!"
      description="작업이 성공적으로 완료되었습니다."
    />
  );
}
```

### API

#### Alert Props

| Property     | Type                                          | Default    | Description                |
| ------------ | --------------------------------------------- | ---------- | -------------------------- |
| type         | `'success' \| 'error' \| 'warning' \| 'info'` | `'info'`   | Alert 타입                 |
| showIcon     | `boolean`                                     | `false`    | 아이콘 표시 여부           |
| closable     | `boolean`                                     | `false`    | 닫기 가능 여부             |
| onClose      | `() => void`                                  | -          | Alert가 닫힐 때 콜백       |
| title        | `ReactNode`                                   | -          | Alert 제목                 |
| description  | `ReactNode`                                   | -          | Alert 설명                 |
| banner       | `boolean`                                     | `false`    | 배너로 표시 여부           |
| size         | `'small' \| 'medium' \| 'large'`              | `'medium'` | Alert 크기                 |
| injectStyles | `boolean`                                     | `true`     | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                                      | -          | 추가 CSS 클래스            |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Alert` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcAlert`: 전체 컨테이너
> - `.hcAlertIcon`: 아이콘 요소
> - `.hcAlertContent`: 텍스트 콘텐츠(제목+설명) 컨테이너
> - `.hcAlertTitle`: 제목 요소
> - `.hcAlertDesc`: 설명 요소
> - `.hcAlertClose`: 닫기 버튼 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customWrapper = style({});

// Alert 배경 및 테두리 커스텀
globalStyle(`${customWrapper} .hcAlert`, {
  backgroundColor: '#fff5f5',
  border: '1px solid #feb2b2',
  padding: '1.5rem',
});

// 제목 폰트 스타일 변경
globalStyle(`${customWrapper} .hcAlertTitle`, {
  color: '#c53030',
  fontSize: '1.1rem',
});
```
