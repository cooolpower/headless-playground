# Radio

라디오 버튼 컴포넌트입니다. 여러 옵션 중 하나만 선택할 수 있습니다.

Radio는 여러 옵션 중 하나만 선택할 수 있는 컴포넌트입니다. RadioGroup을 사용하여 여러 Radio를 그룹화하여 관리할 수 있습니다.

### 사용법

```tsx
import { Radio, RadioGroup } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [value, setValue] = useState('option1');

  return (
    <RadioGroup value={value} onChange={setValue}>
      <Radio value="option1">Option 1</Radio>
      <Radio value="option2">Option 2</Radio>
      <Radio value="option3">Option 3</Radio>
    </RadioGroup>
  );
}
```

### API Reference

#### Radio Props

| Property       | Type                                | Default    | Description                |
| -------------- | ----------------------------------- | ---------- | -------------------------- |
| value          | `string \| number`                  | -          | Radio 값                   |
| checked        | `boolean`                           | -          | 제어되는 선택 상태         |
| defaultChecked | `boolean`                           | `false`    | 기본 선택 상태             |
| disabled       | `boolean`                           | `false`    | 비활성화 여부              |
| name           | `string`                            | -          | Radio 그룹 이름            |
| onChange       | `(value: string \| number) => void` | -          | 값 변경 핸들러             |
| size           | `'small' \| 'medium' \| 'large'`    | `'medium'` | 크기                       |
| injectStyles   | `boolean`                           | `true`     | 기본 스타일(CSS) 주입 여부 |
| className      | `string`                            | -          | 추가 CSS 클래스            |
| children       | `ReactNode`                         | -          | 라벨 콘텐츠                |

#### RadioGroup Props

| Property     | Type                                | Default    | Description                |
| ------------ | ----------------------------------- | ---------- | -------------------------- |
| value        | `string \| number`                  | -          | 제어된 값                  |
| defaultValue | `string \| number`                  | -          | 기본값                     |
| disabled     | `boolean`                           | `false`    | 전체 비활성화 여부         |
| name         | `string`                            | -          | Radio 그룹 이름            |
| onChange     | `(value: string \| number) => void` | -          | 값 변경 핸들러             |
| size         | `'small' \| 'medium' \| 'large'`    | `'medium'` | 크기                       |
| injectStyles | `boolean`                           | `true`     | 기본 스타일(CSS) 주입 여부 |
| className    | `string`                            | -          | 추가 CSS 클래스            |
| children     | `ReactNode`                         | -          | Radio 컴포넌트들           |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Radio` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcRadioGroup`: 여러 라디오 버튼을 감싸는 컨테이너
> - `.hcRadioLabel`: 실제 라디오 버튼과 텍스트를 감싸는 레이블 요소
> - `.hcRadioCircle`: 바깥쪽 원 요소
> - `.hcRadioDot`: 안쪽 체크 표시 원 요소
> - `.hcRadioText`: 라디오 버튼 옆의 텍스트

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customRadioWrapper = style({});

// 라디오 버튼 원형 스타일 커스텀
globalStyle(`${customRadioWrapper} .hcRadioCircle`, {
  width: '24px',
  height: '24px',
  border: '2px solid #cbd5e0',
  backgroundColor: 'white',
});

// 체크된 상태의 스타일 커스텀
globalStyle(`${customRadioWrapper} .hcRadioCircle[data-checked="true"]`, {
  borderColor: '#3182ce',
});

globalStyle(`${customRadioWrapper} .hcRadioDot`, {
  backgroundColor: '#3182ce',
  width: '12px',
  height: '12px',
});

// 텍스트 스타일 커스텀
globalStyle(`${customRadioWrapper} .hcRadioText`, {
  fontSize: '1.1rem',
  color: '#4a5568',
});
```
