# Flip Countdown

Flip 애니메이션으로 카운트다운을 표시하는 컴포넌트입니다. 내부적으로 `Countdown`의 값을 사용해 각 자리(숫자) 변경 시에만 플립 애니메이션이 발생하도록 구현되어 있습니다.

### 사용법

```tsx
import { FlipCountdown } from '@cooolpower/headless-ui';

export function MyComponent() {
  const targetTime = Date.now() + 10 * 1000; // 10초 후
  return <FlipCountdown targetTime={targetTime} />;
}
```

### API Reference

| Property     | Type                                           | Default      | Description                                                                  |
| ------------ | ---------------------------------------------- | ------------ | ---------------------------------------------------------------------------- |
| targetTime   | `number \| Date`                               | -            | 목표 시간 (타임스탬프 또는 Date)                                             |
| active       | `boolean`                                      | `true`       | 활성화 여부                                                                  |
| mode         | `'time' \| 'number'`                           | `'time'`     | 표시 모드 (`time`: 일/시/분/초, `number`: 총 초)                             |
| format       | `string`                                       | `'HH:mm:ss'` | `mode="time"`에서 사용할 포맷 (예: `HH:mm:ss`, `mm:ss`, `ss`, `DD:HH:mm:ss`) |
| minDigits    | `number`                                       | `1`          | `mode="number"`에서 최소 자릿수(왼쪽 0 padding)                              |
| labels       | `Partial<{days,hours,minutes,seconds,number}>` | -            | 단위 라벨 커스터마이징                                                       |
| digitSize    | `'sm' \| 'md' \| 'lg' \| 'xl'`                 | `'md'`       | 한 자리(카드) 크기                                                           |
| injectStyles | `boolean`                                      | `true`       | 기본 스타일(`<style>{flipCountdownCss}</style>`) 자동 주입 여부              |
| className    | `string`                                       | -            | 추가 CSS 클래스                                                              |
| children     | `ReactNode`                                    | -            | 추가 콘텐츠 (컴포넌트 하단에 렌더링)                                         |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`FlipCountdown` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcFlipRoot`: 전체 카운트다운 루트 컨테이너
> - `.hcFlipUnit`: 개별 단위(일, 시, 분, 초) 컨테이너
> - `.hcFlipLabel`: 단위 이름 라벨 (Days, Hours 등)
> - `.hcFlipDigits`: 숫자를 감싸는 영역
> - `.hcFlipDigitContainer`: 개별 숫자 카드의 3D 투영 영역
> - `.hcFlipDigit`: 실제 플립 카드의 외형
> - `.hcFlipTop`, `.hcFlipBottom`: 카드 상/하단 고정부
> - `.hcFlipTopFlip`, `.hcFlipBottomFlip`: 애니메이션되는 플립 조각

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customFlipWrapper = style({});

// 플립 카드 배경 및 텍스트 색상 커스텀
globalStyle(
  `${customFlipWrapper} .hcFlipTop, ${customFlipWrapper} .hcFlipBottom, ${customFlipWrapper} .hcFlipTopFlip, ${customFlipWrapper} .hcFlipBottomFlip`,
  {
    backgroundColor: '#1e293b',
    color: '#38bdf8',
  },
);

// 라벨 스타일 커스텀
globalStyle(`${customFlipWrapper} .hcFlipLabel`, {
  color: '#94a3b8',
  fontWeight: 'bold',
  marginTop: '8px',
});

// 구분선 스타일
globalStyle(`${customFlipWrapper} .hcFlipDivider`, {
  backgroundColor: 'rgba(255, 255, 255, 0.1)',
});
```

###기본 스타일(CSS) (Inject Styles)
