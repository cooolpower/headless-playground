# Input Number

Input Number 컴포넌트는 숫자만 입력할 수 있는 입력 필드를 제공하는 Headless 컴포넌트입니다. 증가/감소 버튼과 키보드 화살표 키를 지원합니다.

## 기본 사용법

Input Number 컴포넌트를 사용하여 숫자 입력 필드를 만들 수 있습니다.

### 기본 Input Number (인터랙티브)

<DemoInputNumberBasicProvider>
  <Playground
    preview={<DemoInputNumberBasicWithControls />}
    controls={<DemoInputNumberBasicControls />}
  />
</DemoInputNumberBasicProvider>

<Playground preview={<DemoInputNumberBasic />} />

```tsx
import { InputNumber } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [value, setValue] = useState<number | undefined>(0);

  return (
    <InputNumber
      value={value}
      onChange={setValue}
      placeholder="Enter a number"
    />
  );
}
```

## Min and Max

최소값과 최대값을 설정할 수 있습니다.

<Playground preview={<DemoInputNumberMinMax />} />

```tsx

```

## Step

증가/감소 단위를 설정할 수 있습니다.

```tsx

```

## API

### InputNumber Props

| Property     | Description                                 | Type                                   | Default    |
| ------------ | ------------------------------------------- | -------------------------------------- | ---------- |
| value        | 제어된 값                                   | `number`                               | -          |
| defaultValue | 기본값                                      | `number`                               | -          |
| onChange     | 값 변경 핸들러                              | `(value: number \| undefined) => void` | -          |
| onFocus      | 포커스 핸들러                               | `(e: React.FocusEvent) => void`        | -          |
| onBlur       | 블러 핸들러                                 | `(e: React.FocusEvent) => void`        | -          |
| min          | 최소값                                      | `number`                               | -          |
| max          | 최대값                                      | `number`                               | -          |
| step         | 증가/감소 단위                              | `number`                               | `1`        |
| precision    | 소수점 자릿수                               | `number`                               | -          |
| disabled     | 비활성화 여부                               | `boolean`                              | `false`    |
| readonly     | 읽기 전용 여부                              | `boolean`                              | `false`    |
| placeholder  | 플레이스홀더 텍스트                         | `string`                               | -          |
| size         | Input Number 크기                           | `'small' \| 'medium' \| 'large'`       | `'medium'` |
| controls     | 증가/감소 버튼 표시 여부                    | `boolean`                              | `true`     |
| keyboard     | 키보드 화살표 키 활성화 여부                | `boolean`                              | `true`     |
| stringMode   | 문자열 모드 (정밀도 유지)                   | `boolean`                              | `false`    |
| className    | 추가 CSS 클래스                             | `string`                               | -          |
| injectStyles | 기본 스타일을 컴포넌트가 직접 주입할지 여부 | `boolean`                              | `true`     |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`InputNumber` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcInputNumber`: 전체 컨테이너 요소
> - `.hcInputNumberInput`: 실제 숫자가 입력되는 `input` 요소
> - `.hcInputNumberButton`: 증가/감소 버튼 요소

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customInputNumberWrapper = style({});

// 컨테이너 스타일 커스텀
globalStyle(`${customInputNumberWrapper} .hcInputNumber`, {
  border: '2px solid #ed8936',
  borderRadius: '0',
  backgroundColor: '#fffaf0',
});

// 입력 필드 텍스트 정렬 및 스타일
globalStyle(`${customInputNumberWrapper} .hcInputNumberInput`, {
  color: '#c05621',
  fontWeight: 'bold',
  fontSize: '1.2rem',
});

// 버튼 디자인 커스텀
globalStyle(`${customInputNumberWrapper} .hcInputNumberButton`, {
  backgroundColor: '#ed8936',
  color: 'white',
});

globalStyle(`${customInputNumberWrapper} .hcInputNumberButton:hover`, {
  backgroundColor: '#dd6b20',
});
```
