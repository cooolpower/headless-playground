# Heatmap

데이터의 분포를 색상으로 표현하는 히트맵 컴포넌트입니다. 시간별, 카테고리별 데이터 밀도를 시각화합니다.

import { Playground } from '@/components/playground/playground';
import { CssSnippet } from '@/components/playground/css-snippet';
import { HeatmapCss } from '@repo/ui';
import {
  DemoHeatmapBasicProvider,
  DemoHeatmapBasicWithControls,
  DemoHeatmapBasicControls,
} from './heatmap.demo';

### 사용법

```tsx
import { Heatmap } from '@repo/ui';

export function MyComponent() {
  return <Heatmap value={data} />;
}
```

### 예제

#### 기본 Heatmap

<DemoHeatmapBasicProvider>
  <Playground
    preview={<DemoHeatmapBasicWithControls />}
    controls={<DemoHeatmapBasicControls />}
  />
</DemoHeatmapBasicProvider>

```tsx
<Heatmap defaultValue={[]} />
```

**Note:** 이 컴포넌트는 현재 개발 중입니다. 향후 업데이트에서 데이터 시각화, 색상 스케일, 인터랙션 등의 기능이 추가될 예정입니다.

### API

#### Heatmap Props

| Property     | Description           | Type                   | Default |
| ------------ | --------------------- | ---------------------- | ------- |
| value        | 제어된 값             | `any`                  | -       |
| defaultValue | 기본값                | `any`                  | -       |
| onChange     | 값 변경 핸들러        | `(value: any) => void` | -       |
| disabled     | 비활성화 여부         | `boolean`              | `false` |
| className    | 추가 CSS 클래스       | `string`               | -       |
| injectStyles | 기본 스타일 주입 여부 | `boolean`              | `true`  |
| children     | 커스텀 콘텐츠         | `React.ReactNode`      | -       |
