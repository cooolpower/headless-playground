# QR Code

QR 코드를 생성하고 표시하는 컴포넌트입니다. 텍스트나 URL을 QR 코드로 변환할 수 있습니다.

### 사용법

```tsx
import { QrCode } from '@cooolpower/headless-ui';

export function MyComponent() {
  return <QrCode value="https://example.com" />;
}
```

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`QrCode` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcQrCode`: 전체 QR 코드 컨테이너
> - `.hcQrCodeContent`: QR 코드 콘텐츠 영역
> - `.hcQrCodeSvg`: QR 코드 SVG를 감싸는 영역
> - `.hcQrCodePlaceholder`: 로딩 중이나 값이 없을 때 표시되는 플레이스홀더

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customQrCodeWrapper = style({});

// QR 코드 컨테이너 스타일 커스텀
globalStyle(`${customQrCodeWrapper} .hcQrCode`, {
  padding: '2rem',
  backgroundColor: '#f8fafc',
  border: '2px dashed #cbd5e0',
  borderRadius: '24px',
});

// QR 코드 SVG 스타일
globalStyle(`${customQrCodeWrapper} .hcQrCodeSvg svg`, {
  filter: 'drop-shadow(0 4px 6px rgba(0, 0, 0, 0.1))',
});

// 로딩 상태 스타일
globalStyle(`${customQrCodeWrapper} .hcQrCode[data-loading="true"]`, {
  filter: 'blur(2px)',
  opacity: 0.7,
});
```
