# Textarea

여러 줄 텍스트 입력을 위한 텍스트 영역 컴포넌트입니다.

### 사용법

```tsx
import { Textarea } from '@repo/ui';

export function MyTextarea() {
  return <Textarea placeholder="텍스트를 입력하세요..." />;
}
```

### API

#### Textarea Props

| Property             | Type                                                                      | Default      | Description                            |
| -------------------- | ------------------------------------------------------------------------- | ------------ | -------------------------------------- |
| size                 | `'custom' \| 'small' \| 'medium' \| 'large'`                              | `'medium'`   | 텍스트 영역 크기                       |
| disabled             | `boolean`                                                                 | `false`      | 비활성화 여부                          |
| readonly             | `boolean`                                                                 | `false`      | 읽기 전용 여부                         |
| placeholder          | `string`                                                                  | -            | 플레이스홀더 텍스트                    |
| value                | `string`                                                                  | -            | 제어된 값                              |
| defaultValue         | `string`                                                                  | `''`         | 기본값                                 |
| onChange             | `(value: string, event?: React.ChangeEvent<HTMLTextAreaElement>) => void` | -            | 값 변경 핸들러                         |
| onFocus              | `(event: React.FocusEvent<HTMLTextAreaElement>) => void`                  | -            | 포커스 핸들러                          |
| onBlur               | `(event: React.FocusEvent<HTMLTextAreaElement>) => void`                  | -            | 블러 핸들러                            |
| onKeyDown            | `(event: React.KeyboardEvent<HTMLTextAreaElement>) => void`               | -            | 키 다운 핸들러                         |
| onClick              | `(event: React.MouseEvent<HTMLTextAreaElement>) => void`                  | -            | 클릭 핸들러                            |
| clearable            | `boolean`                                                                 | `false`      | 지우기 버튼 표시 여부                  |
| maxLength            | `number`                                                                  | -            | 최대 길이                              |
| minLength            | `number`                                                                  | -            | 최소 길이                              |
| required             | `boolean`                                                                 | `false`      | 필수 여부                              |
| autoComplete         | `string`                                                                  | -            | 자동완성 속성                          |
| autoFocus            | `boolean`                                                                 | `false`      | 자동 포커스 여부                       |
| name                 | `string`                                                                  | -            | 텍스트 영역 name 속성                  |
| id                   | `string`                                                                  | -            | 텍스트 영역 id 속성                    |
| rows                 | `number`                                                                  | -            | 행 수                                  |
| cols                 | `number`                                                                  | -            | 열 수                                  |
| resize               | `'none' \| 'both' \| 'horizontal' \| 'vertical'`                          | `'vertical'` | 크기 조절 방향                         |
| autoResize           | `boolean`                                                                 | `false`      | 텍스트 입력 시 자동으로 높이 조절 여부 |
| injectStyles         | `boolean`                                                                 | `true`       | 기본 스타일(CSS) 주입 여부             |
| className            | `string`                                                                  | -            | 추가 CSS 클래스                        |
| textareaClassName    | `string`                                                                  | -            | 텍스트 영역 요소 CSS 클래스            |
| textareaStyle        | `React.CSSProperties`                                                     | -            | 텍스트 영역 요소 인라인 스타일         |
| clearButtonClassName | `string`                                                                  | -            | 지우기 버튼 CSS 클래스                 |
| style                | `React.CSSProperties`                                                     | -            | 컨테이너 인라인 스타일                 |

### 특징 (Features)

- **다양한 크기**: small, medium, large 크기 지원
- **상태 관리**: disabled, readonly 상태 지원
- **지우기 기능**: clearable 옵션으로 지우기 버튼 표시
- **길이 제한**: maxLength, minLength로 입력 길이 제한
- **크기 조절**: resize 옵션으로 크기 조절 방향 제어
- **자동 크기 조절**: autoResize 옵션으로 텍스트 입력 시 자동으로 높이 조절
- **접근성**: ARIA 속성 및 키보드 네비게이션 지원

### 접근성 (Accessibility)

- 키보드로 모든 기능 접근 가능
- 스크린 리더 지원
- 적절한 ARIA 속성 사용
- 포커스 관리

### 스타일링 (Styling)

Textarea 컴포넌트는 `globals.css`의 CSS 변수를 사용하여 스타일이 적용됩니다. 커스텀 스타일이 필요한 경우 `textareaStyle` prop을 사용할 수 있습니다.

### 모범 사례 (Best Practices)

- 긴 텍스트 입력이 필요한 경우 Textarea 사용
- 사용자에게 입력 길이 제한이 있는 경우 `maxLength` 사용
- 폼 제출 시 필수 입력인 경우 `required` 사용
- 크기 조절이 불필요한 경우 `resize="none"` 사용
