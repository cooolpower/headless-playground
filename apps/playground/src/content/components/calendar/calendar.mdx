# Calendar

날짜를 선택하고 표시하는 캘린더 컴포넌트입니다. 월별, 연도별 뷰를 지원하며 날짜 선택 기능을 제공합니다.

### Usage

```tsx
import { Calendar } from '@cooolpower/headless-ui';

export function MyComponent() {
  const [date, setDate] = useState(new Date());

  return <Calendar value={date} onChange={setDate} />;
}
```

### API Reference

#### Calendar Props

| Property           | Type                                                | Default   | Description              |
| ------------------ | --------------------------------------------------- | --------- | ------------------------ |
| value              | `Date`                                              | -         | 제어된 선택 날짜         |
| defaultValue       | `Date`                                              | -         | 기본 선택 날짜           |
| onChange           | `(date: Date) => void`                              | -         | 날짜 변경 콜백           |
| onSelect           | `(date: Date) => void`                              | -         | 날짜 선택 콜백           |
| onPanelChange      | `(date: Date, mode: CalendarMode) => void`          | -         | 패널 변경 콜백           |
| mode               | `'month' \| 'year'`                                 | `'month'` | 표시 모드                |
| disabledDate       | `(date: Date) => boolean`                           | -         | 특정 날짜 비활성화 함수  |
| dateFullCellRender | `(date: Date) => ReactNode`                         | -         | 전체 날짜 셀 커스텀 렌더 |
| dateCellRender     | `(date: Date) => ReactNode`                         | -         | 날짜 셀 커스텀 렌더      |
| monthCellRender    | `(date: Date) => ReactNode`                         | -         | 월 셀 커스텀 렌더        |
| headerRender       | `(config: CalendarHeaderRenderConfig) => ReactNode` | -         | 헤더 커스텀 렌더         |
| fullscreen         | `boolean`                                           | `true`    | 전체 화면 표시 여부      |
| className          | `string`                                            | -         | 추가 CSS 클래스          |
| children           | `ReactNode`                                         | -         | 커스텀 콘텐츠            |
| injectStyles       | `boolean`                                           | `true`    | 기본 스타일 주입 여부    |

### 커스텀 스타일링 (Custom Styling)

`injectStyles={false}`로 설정하면 라이브러리의 기본 스타일이 주입되지 않습니다. 이 경우 사용자가 외부 CSS나 CSS-in-JS를 통해 직접 스타일을 정의해야 합니다.

`Calendar` 컴포넌트는 내부적으로 다음과 같은 고정 클래스명을 제공합니다.

> [!NOTE]
> **제공되는 내부 클래스:**
>
> - `.hcCalendar`: 전체 캘린더 컨테이너
> - `.hcCalendarHeader`: 헤더 영역
> - `.hcCalendarNav`: 네비게이션 버튼 그룹
> - `.hcCalendarNavButton`: 이전/다음 이동 버튼
> - `.hcCalendarHeaderLabel`: 현재 연/월 표시 레이블
> - `.hcCalendarBody`: 달력 그리드 영역
> - `.hcCalendarWeekdays`: 요일 표시 영역
> - `.hcCalendarWeekday`: 개별 요일 요소
> - `.hcCalendarGrid`: 날짜 그리드
> - `.hcCalendarCell`: 개별 날짜 셀
> - `.hcCalendarCellButton`: 날짜 선택 버튼

#### Vanilla Extract 예제

```typescript
import { style, globalStyle } from '@vanilla-extract/css';

export const customCalendarWrapper = style({});

// 캘린더 헤더 스타일 커스텀
globalStyle(`${customCalendarWrapper} .hcCalendarHeader`, {
  backgroundColor: '#f7fafc',
  borderBottom: '2px solid #e2e8f0',
  padding: '1rem',
});

// 날짜 셀 스타일 커스텀
globalStyle(`${customCalendarWrapper} .hcCalendarCellButton`, {
  borderRadius: '8px',
  transition: 'all 0.2s',
});

// 선택된 날짜 스타일
globalStyle(
  `${customCalendarWrapper} .hcCalendarCell[data-selected="true"] .hcCalendarCellButton`,
  {
    backgroundColor: '#38a169',
    color: 'white',
  },
);

// 오늘 날짜 스타일
globalStyle(
  `${customCalendarWrapper} .hcCalendarCell[data-today="true"] .hcCalendarCellButton`,
  {
    color: '#38a169',
    fontWeight: 'bold',
    textDecoration: 'underline',
  },
);
```
