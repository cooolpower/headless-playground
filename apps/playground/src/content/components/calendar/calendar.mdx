# Calendar

날짜를 선택하고 표시하는 캘린더 컴포넌트입니다. 월별, 연도별 뷰를 지원하며 날짜 선택 기능을 제공합니다.

### Usage

```tsx
import { Calendar } from '@repo/ui';

export function MyComponent() {
  const [date, setDate] = useState(new Date());

  return <Calendar value={date} onChange={setDate} />;
}
```

#### Disabled Dates

<Playground preview={<DemoCalendarDisabledDates />} />

```tsx
<Calendar
  disabledDate={(date) => {
    // 주말 비활성화
    const day = date.getDay();
    return day === 0 || day === 6;
  }}
/>
```

#### Custom Cell Render

<Playground preview={<DemoCalendarCustomCell />} />

```tsx
<Calendar
  dateCellRender={(date) => {
    // 특정 날짜에 표시할 내용
    if (date.getDate() === 15) {
      return <span style={{ color: 'red' }}>●</span>;
    }
    return null;
  }}
/>
```

### API Reference

#### Calendar Props

| Property           | Type                                                | Default   | Description              |
| ------------------ | --------------------------------------------------- | --------- | ------------------------ |
| value              | `Date`                                              | -         | 제어된 선택 날짜         |
| defaultValue       | `Date`                                              | -         | 기본 선택 날짜           |
| onChange           | `(date: Date) => void`                              | -         | 날짜 변경 콜백           |
| onSelect           | `(date: Date) => void`                              | -         | 날짜 선택 콜백           |
| onPanelChange      | `(date: Date, mode: CalendarMode) => void`          | -         | 패널 변경 콜백           |
| mode               | `'month' \| 'year'`                                 | `'month'` | 표시 모드                |
| disabledDate       | `(date: Date) => boolean`                           | -         | 특정 날짜 비활성화 함수  |
| dateFullCellRender | `(date: Date) => ReactNode`                         | -         | 전체 날짜 셀 커스텀 렌더 |
| dateCellRender     | `(date: Date) => ReactNode`                         | -         | 날짜 셀 커스텀 렌더      |
| monthCellRender    | `(date: Date) => ReactNode`                         | -         | 월 셀 커스텀 렌더        |
| headerRender       | `(config: CalendarHeaderRenderConfig) => ReactNode` | -         | 헤더 커스텀 렌더         |
| fullscreen         | `boolean`                                           | `true`    | 전체 화면 표시 여부      |
| className          | `string`                                            | -         | 추가 CSS 클래스          |
| children           | `ReactNode`                                         | -         | 커스텀 콘텐츠            |
| injectStyles       | `boolean`                                           | `true`    | 기본 스타일 주입 여부    |
