# Introduction

Headless UI Playground에 오신 것을 환영합니다! 🎉

이 프로젝트는 **스타일에 얽매이지 않고 (Headless) 기능만 완벽히 구현된 UI 컴포넌트**들을 모아놓은 라이브러리 및 데모 사이트입니다.

## 주요 특징

### 🎨 테마 및 커스텀 자유도

모든 컴포넌트는 CSS 변수(Variables)를 기반으로 디자인되어 있습니다.
사용자는 구조나 스크립트 수정 없이 오직 CSS 변수 조작만으로 완전히 다른 테마(다크 모드 등)와 프리미엄 디자인을 구성할 수 있습니다.

### ⚡ 프레임워크 독립적인 순수 스타일

특정 CSS-in-JS 라이브러리에 종속되지 않습니다. 핵심 스타일은 바닐라 CSS 텍스트로 내보내어지며(Vanilla Extract 활용), 원한다면 `injectStyles={false}`로 끄고 사용자만의 CSS를 입혀 완전히 새롭게 탈바꿈할 수 있습니다.

### 🛠 타입 안전성과 확장성

React(Next.js) 환경에 최적화되었으며 모든 기능이 TypeScript로 작성되어 훌륭한 자동 완성(IntelliSense)을 제공합니다.
각 컴포넌트의 로직은 커스텀 훅(`useButton`, `useTimeline` 등)으로 깔끔하게 분리되어 있어, UI만 다르게 그리고 로직은 가져다 쓰는 완벽한 Headless 패턴을 지원합니다.
