# Installation

`@cooolpower/headless-ui` 패키지를 설치하고 프로젝트에 적용하는 방법을 안내합니다.
공식 패키지 정보는 [npm 패키지 페이지](https://www.npmjs.com/package/@cooolpower/headless-ui)에서 확인하실 수 있습니다.

> [!NOTE]
> 이 문서의 예제 코드에서는 `@cooolpower/headless-ui`를 사용합니다.
> 만약 이 프로젝트(Monorepo) 내부에서 개발 중이라면 `@cooolpower/headless-ui`로 바꿔서 사용하셔도 동일합니다.

## 1. 패키지 설치

NPM, Yarn, 혹은 PNPM을 사용하여 패키지를 설치합니다.

```bash
# pnpm
pnpm add @cooolpower/headless-ui

# npm
npm install @cooolpower/headless-ui

# yarn
yarn add @cooolpower/headless-ui
```

## 2. 스타일 적용하기 (필수)

UI 컴포넌트들은 **사전 정의된 CSS 변수(Variables)** 를 통해 디자인과 색상을 결정합니다.
아래 두 가지 방법 중 하나를 선택하여 스타일을 적용합니다.

### 방법 A: 통합 CSS 임포트 (권장)

패키지에서 제공하는 `styles.css`를 임포트하면 **테마 변수 + 모든 컴포넌트 스타일**이 한 번에 적용됩니다.
페이지 로드 시 스타일이 즉시 반영되므로 **FOUC(깜빡임 현상)가 발생하지 않습니다.**

```tsx
// app/layout.tsx
import '@cooolpower/headless-ui/styles.css';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  );
}
```

> [!TIP]
> 이 방법을 사용하면 각 컴포넌트의 `injectStyles` prop을 `false`로 설정해도 스타일이 정상 적용됩니다.

### 방법 B: 테마 변수만 주입 (동적 스타일)

테마 변수만 직접 주입하고, 컴포넌트 스타일은 각 컴포넌트가 렌더링될 때 동적으로 삽입하는 방식입니다.
번들 크기를 최소화하고 싶을 때 유용합니다.

```tsx
// app/layout.tsx
import { defaultThemeCss } from '@cooolpower/headless-ui';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ko">
      <head>
        <style dangerouslySetInnerHTML={{ __html: defaultThemeCss }} />
      </head>
      <body>{children}</body>
    </html>
  );
}
```

> [!NOTE]
> 방법 B는 컴포넌트가 클라이언트에서 마운트될 때 스타일을 삽입하므로, 초기 로드 시 짧은 FOUC가 발생할 수 있습니다.

## 3. 사용할 컴포넌트 불러오기

이제 컴포넌트를 자유롭게 가져와서 사용하시면 됩니다. 각각의 컴포넌트는 자신이 사용될 때 자동으로 필요한 스타일을 브라우저에 동적으로 삽입하므로 번거롭지 않습니다.

```tsx
import { Button, Toast, Timeline } from '@cooolpower/headless-ui';

export function MyComponent() {
  return (
    <div>
      <Button color="primary">Welcome</Button>
    </div>
  );
}
```

## 컴포넌트 스타일 완전 커스텀하기 (Optional)

만약 패키지에 포함된 고정 스타일을 전혀 쓰고 싶지 않다면, 각 컴포넌트에 포함된 `injectStyles={false}` props를 전달하면 됩니다. 그 후, 클래스 이름에 맞춰 자신만의 CSS를 연결하여 순수한 커스텀 요소로 활용할 수도 있습니다!

```tsx
// 기본으로 깔리는 ButtonCss를 브라우저에 심지 않고, 내가 만든 CSS만 씁니다.
<Button injectStyles={false} className="my-custom-button-class">
  Custom Button
</Button>
```
